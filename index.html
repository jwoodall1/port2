Text Generator

A web-based project that dynamically generates scrolling sci‑fi stories using a Star Wars–style crawl effect. It randomly combines preset narrative templates with words from a JSON file and features an animated starfield background.

Documentation

Find The Ball

An interactive p5.js game where you help a playful dog locate its tennis ball in a room full of interactive zones. Click carefully to avoid too many "Oopsies" and win the game.

Documentation

Coin Drop

A hand-gesture controlled game that uses ml5.js and p5.js. With your webcam, control a bucket to catch falling coins while dodging bombs. Finger-triggered buttons start the game and toggle features, challenging you to beat your high score.
